ChartStack by Keen IO

ChartStack is an open-source JavaScript library that extends HTML with new, simple yet powerful charting elements like and . It allows creating complex charts using static or data-driven data with as minimal coding as possible. Despite it's simplicity ChartStack charts support animations and other advanced features and can be styled and tweaked with CSS.

No charting library lock-in

Under the hood, ChartStack relies on a charting library but is not fixed to any particular charting library. It uses NVD3 charting library (built on D3) by default but also currently support Google Charts with more charting libraries to come.

Includes easy transformation of your data format

We recognize that everybody collects data in different ways and every charting library has it's own arbitrary data format for displaying that data, that's why ChartStack comes with easy methods to normalize incoming data to one unified format. Though there is no real standard for displaying data in charts, once you've setup the normalization plugin for your data source, all charts just work. We've already included a normalization plugin for Keen IO data sources and the community is invited to add theirs in the ChartStack Github repo.

Change the charting library but not your data

Because Chartstack understands your data format it means switching to a different supported charting library is as easy as a text change without having to change your data to support the other charting api. That way you can focus on building and styling your chart.

Support for advanced users

Chartstack's goal is to be as simple and flexible while not tying you to one Graphing Library. There are cases however, where one graphing library has features that another doesn't. Chartstack keeps the underlying library accessible so that advanced users can still access those features.

Key benefits:

  • Allows data providers to adapt their data to any charting library.
  • Building your own charts is as easy as including one JS tag on the bottom of the page and putting your or anywhere on the page.
  • Switching graphing libraires doesn't change your embed code [by much]
  • Embed code is easy to understand and edit by developers and designers alike.
  • JS lib exposes graphing library underneath for more complex options.

Key benefit for Keen users

  • Break hard dependency on Google Charts
  • Allows Keen data to work with any charting framework easily and with one standard markup/api.
  • Basis for advanced charting ability (Dashstack on wrapper around Chartstack for Keen-specific interactions.)
  • Supports multiple data consuming methods includes ajax and embedded JSON.
  • Open source and modular so people can build on it.
